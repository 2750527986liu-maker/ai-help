# Claude Desktop CN + DeepSeek 使用说明

## 一键安装

1. 双击 `一键安装.bat`
2. 按提示输入安装路径（直接回车默认安装到当前目录，也可以输入 `D:\Claude` 等自定义路径）
3. 等待自动下载安装完成
4. 桌面会出现 "Claude + DeepSeek" 快捷方式

## 启动使用

- 双击 `launch.bat` 或桌面快捷方式
- CCSwitch 会自动启动（DeepSeek API 网关）
- Claude Desktop CN 会自动启动
- 在 Claude Desktop CN 中正常使用即可，API 请求会通过 DeepSeek 处理

## 首次使用配置

启动 Claude Desktop CN 后：

1. 登录你的 Claude 账号
2. CCSwitch 会自动将 Claude API 请求转发到 DeepSeek
3. 无需翻墙，直接使用

## DeepSeek API 说明

- 官方地址：https://api.deepseek.com
- 可用模型：deepseek-v4-flash、deepseek-v4-pro
- 需要自行注册 DeepSeek 账号并获取 API Key

## 卸载

双击 `uninstall.bat` 即可完全卸载，包括：
- CCSwitch 网关工具
- Claude Desktop CN
- 临时下载文件
- 桌面快捷方式

## 常见问题

**Q: 下载失败怎么办？**
A: 脚本会自动尝试多个镜像源。如果全部失败，请检查网络连接后重试。

**Q: 启动后 Claude Desktop CN 显示连接错误？**
A: 确保 CCSwitch 已启动（检查系统托盘图标）。如果没有，手动运行 `CCSwitch\CCSwitch.exe`。

**Q: 可以安装到 D 盘吗？**
A: 可以。安装时输入自定义路径如 `D:\Claude`，所有文件和快捷方式都会使用绝对路径指向该目录。

**Q: Claude Desktop CN 和官方版有什么区别？**
A: Claude Desktop CN 是针对中国用户的本地化版本，由 Qiao-920 打包，可在国内网络环境下使用。
