# Claude Desktop + DeepSeek 便携版

## 安装

双击 **一键安装.bat**，等待下载完成（约 280MB）。

## 获取 API Key

打开 [platform.deepseek.com](https://platform.deepseek.com) 注册，免费获取。

## 首次配置

双击 **launch.bat**：

CCSwitch → Add Provider → DeepSeek → 粘贴 API Key → 开 Gateway

Claude → Developer → Third-Party Inference → Gateway → `http://127.0.0.1:15721`

## 卸载

双击 **uninstall.bat**
