## Claude Desktop + DeepSeek 便携版（无需翻墙）

一键安装 Claude Desktop + DeepSeek，支持 U 盘便携运行，无需翻墙。

### 快速开始

1. 双击 `一键安装.bat`
2. 等待约 5 分钟下载安装
3. 在 [platform.deepseek.com](https://platform.deepseek.com) 免费获取 API Key
4. 双击 `launch.bat`，配置 CCSwitch 连接 DeepSeek

### 文件说明

| 文件 | 用途 |
|------|------|
| `一键安装.bat` | 安装入口 |
| `install.ps1` | 安装脚本 |
| `使用说明.md` | 中文教程 |

安装后的应用目录（`CCSwitch/`、`ClaudeDesktop/`）不纳入版本管理。
