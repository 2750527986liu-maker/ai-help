# Claude Desktop CN + DeepSeek 一键安装器

无需翻墙，一键安装 Claude Desktop CN + DeepSeek API 网关。

## 功能特点

- 一键安装，新手友好
- 自动下载 Claude Desktop CN（Qiao-920 本地化版本）
- 自动配置 CCSwitch API 网关（DeepSeek）
- 支持自定义安装路径（如 D 盘）
- 自带卸载功能
- 内置多个下载镜像，自动选择最快的源

## 使用方法

1. 下载本项目
2. 双击 `一键安装.bat`
3. 按提示选择安装路径（直接回车默认当前目录）
4. 等待安装完成
5. 使用桌面快捷方式启动

## 组件说明

| 组件 | 来源 | 说明 |
|------|------|------|
| Claude Desktop CN | [Qiao-920/claude-desktop-cn](https://github.com/Qiao-920/claude-desktop-cn) | Claude 桌面端中文版 |
| CCSwitch | [farion1231/CCSwitch](https://github.com/farion1231/CCSwitch) | API 网关工具 |
| DeepSeek | [deepseek.com](https://deepseek.com) | AI 模型服务 |

## 系统要求

- Windows 10/11 (64-bit)
- 网络连接（用于首次下载）
- DeepSeek API Key（需自行注册）
